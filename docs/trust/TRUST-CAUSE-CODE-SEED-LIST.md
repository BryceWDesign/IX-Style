# IX-Style Trust Cause Code Seed List

## Purpose

This document seeds a stable set of trust-transition cause codes for later use
in schemas, evidence, verification, and implementation.

These are seed codes, not the final closed set.

## Notes

These cause codes exist for three reasons:

1. to keep trust transitions machine-readable
2. to support deterministic verification cases
3. to avoid vague evidence like "trust went down for reasons"

A later commit can freeze, expand, or tailor this list.
